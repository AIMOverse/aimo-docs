# AiMo Network - User Guide

## Getting Started

Getting started with AiMo Network is simple - just three steps:

### 1. Visit the Web Console

Go to the [AiMo Network Dashboard](https://aimo.network/account) to access your account.

### 2. Fund Your Wallet

Add USDC to your account through the web console. This will be used for paying AI inference requests.

### 3. Create an API Key

Generate an API key in the dashboard and start making API calls immediately.
